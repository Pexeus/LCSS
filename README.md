## Development

| Scripts            | Info                         |
| ------------------ | ---------------------------- |
| `npm i`            | installing deps              |
| `npm start`        | starting project             |
| `npm run build`    | building project             |
| `npm run lint:js`  | run eslint                   |
| `npm run lint:fix` | run eslint with `--fix` flag |
| `npm run buildInstaller` | create .msi Installer  |
